// Project Timeline

How It Was Built

PHASE 1 — FOUNDATION

🖥️ Ubuntu Server Setup on Repurposed Hardware

Wiped a consumer laptop (Intel i7-1255U, 16GB RAM, Iris Xe integrated graphics) down to a clean Ubuntu 24.04 install, freeing ~290GB by stripping the desktop environment, snap packages, and old backups. Installed Docker CE fresh from the official repo, mounted a 4TB Seagate expansion drive with fstab/udev automount, and set up Tailscale for remote access without exposing anything to the public internet.

Ubuntu 24.04 Docker CE Tailscale exFAT + udev
PHASE 2 — CUSTOM DASHBOARD

🏠 HANK Dashboard — Built From Scratch

Built the entire dashboard by hand: a Node.js/Express backend that talks directly to the Linux system for live stats, and a hand-written HTML/CSS/JS frontend with no framework — every page, modal, and animation coded from zero. Runs as a systemd service, served over HTTPS using a Tailscale-issued certificate.

Node.js Express Vanilla JS/HTML/CSS systemd HTTPS
PHASE 3 — PHOTOS & MEDIA

📸 Immich Photo Library + Plex with Hardware Transcoding

Deployed Immich with separate libraries and admin accounts for each family member so photo collections stay independent. Wired Plex into the dashboard's Movies & TV page and fixed hardware-accelerated transcoding by passing the Iris Xe iGPU into the container (device + group_add) — verified live via transcoder session logs instead of just trusting the settings toggle. Built a from-scratch music player streaming a 710-song library alongside a standalone Navidrome instance.

Immich Plex Intel Quick Sync Navidrome
PHASE 4 — DOCUMENTS, PRINT & SCAN

🖨️ Real Printer/Scanner Integration

Built an in-dashboard document manager with thumbnails, previews, move/delete, and folder creation — plus a full Print/Scan page driving a physical Epson WF-2950: flatbed scanning with resolution and color-mode controls via SANE's eSCL backend, direct CUPS printing with automatic Word-to-PDF conversion, live ink-level monitoring, and a phone-camera "scan a document" flow using the native camera app for full-resolution capture.

SANE / eSCL CUPS + IPP ImageMagick Ghostscript
PHASE 5 — HANK, THE AI ASSISTANT

🎙️ Voice + Chat Assistant Built Into the Dashboard

Integrated an AI assistant (HANK) powered by Groq's Llama 3.3 70B, streaming responses sentence-by-sentence for low-latency replies. Voice synthesis runs through a self-hosted Edge-TTS container with an indexed audio-slot queue guaranteeing in-order playback — including working microphone input on mobile browsers.

Groq / Llama 3.3 Edge-TTS Web Speech API FastAPI
PHASE 6 — MONITORING

📊 System Monitoring & Network Visibility

Added a background monitor that watches CPU temperature (tiered alerts at 85°C/90°C/95°C), RAM, disk usage, and container uptime — pushing instant phone notifications via a self-hosted ntfy instance. Built a live network device scanner using arp-scan, showing every device on the LAN with editable custom names, and a security panel that tracks failed SSH attempts from auth.log.

ntfy arp-scan systeminformation
PHASE 7 — DISASTER RECOVERY

🛠️ Recovered From a Full Frontend Wipe

A Python encoding bug during a file write truncated the entire dashboard frontend to zero bytes. Rebuilt it from scratch using recovered code fragments and reference screenshots, then — having learned that lesson — built a proper disaster-recovery process: a full versioned tar.gz backup of every service config, the systemd unit, and sudoers rules, plus a generated PDF reference of the complete codebase for quick lookup without needing to SSH in.

Incident Recovery tar.gz Backups Documentation

Security & Monitoring

🌐

Tailscale Zero-Config VPN

The dashboard is reachable over HTTPS through Tailscale's encrypted mesh network — no port-forwarding, no public-facing admin surface.

CONNECTED
📈

Failed-Login Monitoring

Parses auth.log in real time for failed SSH attempts, surfaced live in the dashboard's Security Monitor panel.

MONITORING
📱

Tiered Push Alerts

Self-hosted ntfy sends instant phone notifications for CPU temperature thresholds, high RAM/disk usage, and container downtime.

ACTIVE
🔒

HTTPS Everywhere

The dashboard is served exclusively over HTTPS using a certificate issued through Tailscale — never plain HTTP.

ENFORCED
🗺️

Network Device Scanner

Live arp-scan of the LAN shows every connected device by IP and MAC, with editable custom names so nothing is a mystery.

LIVE
📂

Path Traversal Protection

Every file-serving API — Documents, Files, Photos — validates that resolved paths stay within their allowed root before touching disk.

ENFORCED
🔧

Scoped Sudo Permissions

Network scanning runs through a narrowly-scoped sudoers rule for arp-scan only — not a broadly privileged process.

HARDENED
💾

Full Disaster Recovery Backups

Versioned tar.gz archives of every service config, the systemd unit, and sudoers rules — built after a real data-loss incident.
